<h2><a href="https://www.geeksforgeeks.org/problems/maximum-average-subarray5859/1">Maximum Average Subarray</a></h2><h3>Difficulty Level : Difficulty: Easy</h3><hr><div class="problems_problem_content__Xm_eO" style="--text-color: var(--problem-text-color);"><p><span style="font-size: 18px;">Given an array <strong>arr[]</strong> and a positive integer <strong>k</strong>, find the subarray of length k having the <strong>maximum </strong>average value.</span></p>
<p><span style="font-size: 18px;">Return the <strong>starting </strong>index of that subarray.</span></p>
<p><span style="font-size: 18px;">If multiple subarrays have the same maximum average, return the <strong>smallest </strong>starting index.</span></p>
<p><span style="font-size: 18px;"><strong>Examples:</strong></span></p>
<pre><span style="font-size: 18px;"><strong>Input: </strong>k = 4, arr[] = [1, 12, -5, -6, 50, 3]
<strong>Output:</strong> 1
<strong>Explanation:</strong> Maximum average is (12 - 5 - 6 + 50)/4 = 51/4. Therefore answer for this test case is 1.</span></pre>
<pre><span style="font-size: 18px;"><strong>Input: </strong>k = 3, arr[] = [3, -435, 335, 10, -50, 100, 20]
<strong>Output:</strong> 2
<strong>Explanation:</strong> Maximum average is (335 + 10 - 50)/3 = 295/3. Therefore answer for this test case is 2.
</span></pre>
<p><span style="font-size: 18px;"><strong>Constraints</strong><br>1 ≤ k ≤ arr.size() ≤ 10<sup>5<br></sup></span><span style="font-size: 18px;">-10<sup>3</sup> ≤ arr[i] ≤ 10<sup>3</sup></span></p>
